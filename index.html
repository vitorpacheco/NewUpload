NewUpload by vitorpc

Upload Behavior for CakePHP 2.0

An Upload Behavior for CakePHP 2.0. This behavior allows you to optionally store in a database the information about your uploads.

Dependencies

CakePHP 2.0 PHP 5.3

Install

Installation Get a copy of the code on GitHub. Some methods to get code: Download the source code at this link: Download code from the branch master Add it as a submodule of your project: git submodule add git://github.com/vitorpc/NewUpload.git Plugin/NewUpload Make a clone of the project: git clone git://github.com/vitorpc/NewUpload.git Plugin/NewUpload Add to your Config/bootstrap.php the following code: CakePlugin::load('NewUpload'); Usage You must specify the property $actsAs in their model.
